# First C++ Games

Early C++ game projects written during freshman year — a record of first steps into systems programming and game logic.

---

## About

These projects represent the beginning of a programming journey: learning memory management, game loops, input handling, and terminal rendering through hands-on game development in C++.

Topics covered through these projects:
- Basic game loop design
- Terminal/console rendering
- Keyboard input handling
- Collision detection fundamentals
- Score tracking and state management

---

## Build

```bash
g++ -o game main.cpp
./game
```
